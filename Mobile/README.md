# ERP Mobile - تطبيق المخازن ونقاط البيع

تطبيق Flutter لإدارة المخازن ونقاط البيع متوافق مع متطلبات النظام الضريبي المصري (ETA - الفاتورة الإلكترونية).

## المميزات

- **نقاط البيع (POS):**
  - فتح/إغلاق جلسة الكاش
  - مسح الباركود (Scanner) للأصناف
  - بحث وإضافة الأصناف من شبكة المنتجات
  - حساب تلقائي للضريبة (VAT 14%)
  - طرق دفع متعددة (كاش، بطاقة، InstaPay، محفظة، آجل)
  - إرسال الفاتورة لمصلحة الضرائب المصرية (ETA)
  - عرض وطباعة الفواتير

- **المخازن:**
  - عرض الأصناف وأرصدتها
  - عرض رصيد المخزن لكل مخزن
  - تعديل الأرصدة (Stock Adjustment)

## البدء

```bash
flutter pub get
flutter run
```

ثم عدّل عنوان الـ API في `lib/core/config/api_config.dart`:

```dart
static const String baseUrl = 'http://YOUR_SERVER_IP:5000/api';
```

## البنية

- `core/` - الإعدادات والخدمات الأساسية والـ Theme
- `features/auth/` - شاشة تسجيل الدخول
- `features/home/` - الشاشة الرئيسية
- `features/inventory/` - شاشات المخازن
- `features/pos/` - شاشات نقاط البيع
